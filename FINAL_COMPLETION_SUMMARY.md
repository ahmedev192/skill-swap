## 🚀 **TO GET STARTED**

1. **Update configuration** in `appsettings.json`:
   - Connection string
   - JWT key
   - SendGrid API key

2. **Run the application**:
   ```bash
   dotnet restore
   dotnet build
   cd src/SkillSwap.API
   dotnet run
   ```

3. **Access Swagger UI** at the root URL

4. **Start using the API** - everything is ready!
